# Módulo: Colegio

Este módulo se encarga del onboarding de los establecimientos educativos, la creación de perfiles de usuario (administradores, jefes de UTP y docentes), y la gestión de cursos, alumnos y matrículas académicas.

## Tablas Asociadas
- `miutp_colegios`
- `miutp_perfiles`
- `miutp_cursos`
- `miutp_estudiantes`
- `miutp_matriculas`
- `miutp_profesores_cursos`
