# Módulo: AI Agent

Este módulo contiene la lógica para interactuar con la IA de Gemini utilizando el SDK de IA de Vercel. Se encarga de las propuestas de preguntas personalizadas, las actividades de reforzamiento pedagógico y la observabilidad de IA con Langfuse.
