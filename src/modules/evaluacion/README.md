# Módulo: Evaluación

Este módulo gestiona la creación de pruebas y ensayos, pautas de corrección, y la asociación de preguntas a los estándares oficiales del Ministerio de Educación (OA y habilidades).

## Tablas Asociadas
- `miutp_pruebas`
- `miutp_preguntas`
- `miutp_pruebas_aplicadas`
- `miutp_estandares_aprendizaje`
- `miutp_objetivos_aprendizaje`
- `miutp_habilidades`
